Exercício - Kasnar Leonardos

por: Adriana Borges

Critérios tecnológicos utilizadas no projeto:

  • ASP.NET Core e C# para código do lado do servidor de plataforma cruzada
  • Angular e TypeScript para código do lado do cliente
  • Bootstrap para layout e estilo
  • SQLServer para banco de dados
  • Código fonte disponibilizado no Github

O projeto em questão possui documentação do Swagger, que mostrará toda a documentação das APis pela URL / api/documentation que é atualizada automaticamente quando você criado uma nova API.
